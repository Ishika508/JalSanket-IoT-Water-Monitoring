# System Architecture Overview

The system is divided into three layers:

---

## 1️⃣ IoT Layer

- pH, TDS, Turbidity Sensors  
- ESP32 Microcontroller  
- WiFi-based Data Transmission  

---

## 2️⃣ Backend & AI Layer

- Node.js + Express REST API  
- MVC Architecture  
- AI Risk Engine (Anomaly Detection & Risk Scoring)  
- Public Health Data Integration  

---

## 3️⃣ Dashboard & Alert Layer

- Citizen Dashboard  
- Government Monitoring Dashboard  
- Real-time Risk Visualization  
- SMS-based Alerts for Low Connectivity Areas  
- Inspection & Complaint Tracking System
