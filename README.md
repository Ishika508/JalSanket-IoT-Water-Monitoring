# JalSanket – Smart Household Water Quality Monitoring

## Hack-A-Cause | PRERNA 18.0 Final Round

JalSanket is a low-cost IoT-enabled system designed to monitor water quality in household tanks and taps.

The system detects contamination risks using real-time sensor monitoring and AI-assisted analysis.

---

## Problem Statement
Smart Household Tank & Tap Water Quality Monitoring System

Key goals:
- Detect abnormal pH and TDS levels
- Monitor stored water quality
- Provide real-time alerts
- Enable citizen reporting
- Support low-connectivity environments

---

## Proposed Architecture

ESP32 Sensors → Backend API → Database → User Dashboard → Government Dashboard

---

## Tech Stack (Planned)

- ESP32 (IoT Hardware)
- Node.js + Express Backend
- MongoDB Database
- Next.js Dashboard
- AI-based Risk Prediction
